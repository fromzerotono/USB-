# 原理
#打开第1路USB开关不反馈：A0 01 01 A2，继电器会吸合，但不会反馈数据
#关闭第1路USB开关不反馈：A0 01 00 A1，继电器会释放，但不会反馈数据
注意：发送数据后需要time.sleep()，无论是吸合还是断开，不然指令会被快速覆盖

# 需要权限，添加永久权限：
    1. 将当前用户添加到 dialout 组
    sudo usermod -a -G dialout $USER
    2. 重新加载组权限（或重新登录/重启）
    newgrp dialout
    添加后需要重新登录或重启系统才能生效。

# 快速检查命令：
    1.检查设备是否存在
    ls /dev/ttyUSB*
    ls -l /dev/ttyUSB0
    2.检查当前用户所属组
    groups
    3.检查 dialout 组成员
    getent group dialout
    4.查看 USB 设备
    lsusb
